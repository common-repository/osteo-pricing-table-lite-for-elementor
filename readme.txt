== Installation ==

Follow to install this plugin:

1. Upload the plugin files to the `/wp-content/plugins/osteo-pricing-table-for-elementor-lite` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
